## Como rodar o app
1- Para baixar todas as dependencias do app antes de rodar:
```bash
 yarn install
```

2- Rodar o app:
```bash
yarn start
```

3- Pronto! agora é só usar o link abaixo no navegador para vê-la
```bash
http://localhost:8080/
```
